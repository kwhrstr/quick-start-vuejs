# quick-strat-vuejs

Vue.jsのtutorialを行うためのプロジェクト

Vue.jsはCDNパッケージとして取り込む
`<script src="https://unpkg.com/vue@next"></script>`

実装するのは[Vuejs-v3ガイダンスのhtml, css, javascript](https://v3.ja.vuejs.org/guide/introduction.html)
